Struture qui définit les entrées et sorties des fonctions et méthodes

Déclaration

Par exemple on peut déclarer une variable ou une fonction. On déclare à l'aide d'un mot clef (par exemple var let ou const). On peut appeler ultérieurement ce qu'on déclare. Il faut nommer une déclaration. La déclaration est associée à un nom dans lequel seront stockés des données.
par ex : var maVariable;

Paramètres

Synonyme de arguments. Associés à une fonction, il fonctionne comme une variable locale. En informatique, les arguments sont les données traitées par une fonction. Ils sont remplacés par les entrées lors de l'exécution du programme qui utilise cette fonction.
par ex : function maFonction(){
}

Arguments

Synonyme de paramètre. En programmation informatique, un paramètre est une donnée manipulée par une section de code (voir : sous-programme, fonction, méthode) et connue du code appelant cette section.

Récursivité

Quand une fonction fait appelle à elle même.Tout comme pour les boucles, les fonctions récursives vont nous permettre d’exécuter une action en boucle et jusqu’à ce qu’une certaine condition de sortie soit vérifiée.

Pile d’appel

Sens d'éxécution des fonctions qui gère les entrées et les sorties. Mécanisme pour un interpréteur de garder une trace dans un script appelant des multiples fonctions, c'est à dire quelle est la fonction exécutée ? quelles fonctions sont appelées à l'intérieur d'autres fonctions.
schéma

Fonctions anonymes

Une fonction anonyme ne porte pas de nom. Syntaxe raccourcie. Elle sera utilisée à un endroit définit de notre script. Elle peut s'éxécuter en s'auto invoquant, en étant appelée dans une variable ou dans un évènement.
